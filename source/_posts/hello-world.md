## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

More info: [Writing](https://hexo.io/docs/writing.html)

### Run server

``` bash
$ hexo server
```

More info: [Server](https://hexo.io/docs/server.html)

### Generate static files

``` bash
$ hexo generate
```

More info: [Generating](https://hexo.io/docs/generating.html)

### Deploy to remote sites

``` bash
$ hexo deploy
```

More info: [Deployment](https://hexo.io/docs/one-command-deployment.html)

## 新增内容

### Hexo简介

Hexo是一个快速、简洁且高效的博客框架。它使用Markdown（或其他渲染引擎）来解析文章，并且能够快速生成静态网页。

### 优势

- 快速生成：得益于Node.js的强大性能，Hexo可以在几秒内生成数百个页面。
- 支持Markdown：使用熟悉的Markdown语法撰写文章。
- 强大的插件系统：丰富的插件可以扩展Hexo的功能。
- 主题系统：可以轻松更换和自定义主题。

### 常用命令总结

| 命令 | 描述 |
|------|------|
| `hexo new` | 创建新文章 |
| `hexo generate` | 生成静态文件 |
| `hexo server` | 启动本地服务器 |
| `hexo deploy` | 部署到远程站点 |
| `hexo clean` | 清除缓存文件和静态文件 |
